# Crawl freely. Privacy is enforced by the app, not by robots.txt:
#   - Owner data (/api/me/pins, PATCH/DELETE /api/pins) is AUTH-GATED (401/403
#     to anonymous requests) — that gate is the control, not a Disallow.
#   - App/UI pages (pin, manage, edit) and the public coords lookup carry
#     noindex (HTML <meta robots> / JSON X-Robots-Tag) so they stay out of
#     search while remaining crawlable. We deliberately do NOT Disallow them:
#     a Disallow-ed page is never fetched, so its noindex is never seen — the
#     two cancel (see docs/04-process/privacy-seo.md, Tier 2.5).
#   - Only the marketing apex (/) is indexable, and it carries no PII.
User-agent: *
Allow: /
