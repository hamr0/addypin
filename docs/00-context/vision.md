# AddyPin - Product Vision

## What Is AddyPin?

AddyPin is a location sharing service that transforms GPS coordinates into short, memorable links. Users create pins on an interactive map and share them via dual formats:

- **Web links**: `ABC123.addypin.com`
- **Email addresses**: `ABC123@addypin.com`

Send an email to `ABC123@addypin.com` and receive an auto-response with map links for that location across 13+ map applications.

## Live Environments

- **Production**: https://addypin.com
- **Staging**: https://staging.addypin.com

## Core Value Proposition

- **Simple**: Click a map, get a shareable link
- **Universal**: Works with Google Maps, Apple Maps, Waze, and 10+ other map apps
- **Dual format**: Share via web link or email address
- **Privacy-first**: No accounts required, pins without email expire in 72 hours
- **Cost-effective**: Entire infrastructure runs for ~$2.83/month

## Product Boundaries

### What AddyPin Does
- Converts GPS coordinates to short, shareable codes
- Provides links to 13+ map applications
- Sends auto-response emails with location data
- Tracks basic analytics (pin creation, clicks, countries)

### What AddyPin Does Not Do
- Navigation or routing
- Real-time location tracking
- Social features or user profiles
- Paid tiers or subscriptions (currently)
